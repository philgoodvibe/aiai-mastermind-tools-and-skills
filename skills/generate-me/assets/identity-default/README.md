# Primary Identity Photos

This folder holds the five primary identity photos. These original photos are the authority for likeness and outrank generated examples or written descriptions. The folder ships empty on purpose so no personal photos or identity details are included in the template.

Install exactly one image for each slot, using `.jpeg` or `.png` consistently or as appropriate:

1. `01-primary-face.jpeg` or `01-primary-face.png`
   Authority for clear facial structure, feature spacing, fine features, skin texture and hairline.
2. `02-upper-body.jpeg` or `02-upper-body.png`
   Primary overall authority for likeness, posture, shoulders, build and head-to-torso scale.
3. `03-proportions.jpeg` or `03-proportions.png`
   Authority for body proportions, stance, relative scale and limb-to-torso balance.
4. `04-expression-open-smile.jpeg` or `04-expression-open-smile.png`
   Authority for broad open-smile behavior, visible teeth, lip movement and cheek lift.
5. `05-secondary-face.jpeg` or `05-secondary-face.png`
   Authority for confirming stable facial geometry, hairline and feature shape from a second view.

Use only original camera photos. Never place AI-generated or AI-retouched images here.
