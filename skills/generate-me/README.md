# Generate Me

Generate Me creates images of the skill owner while preserving their real facial identity across photoreal, illustrated, cartoon, cinematic and stylized scenes. It is a blank personal template, so it ships with no photos and no personal identity details.

To install and personalize it, point a coding AI agent at this folder and ask the agent to follow [SETUP.md](SETUP.md). The agent will inspect the owner's supplied reference photos, write the factual identity profile and check the available generation path. Do not copy setup steps from elsewhere, use the linked guide as the source of truth.

The owner should plan to supply about five suitable photos and about ten minutes. A coding AI agent does the customization for them, so they do not need to edit profile files by hand.

The setup can be performed with Claude Code, Codex or Gemini.
